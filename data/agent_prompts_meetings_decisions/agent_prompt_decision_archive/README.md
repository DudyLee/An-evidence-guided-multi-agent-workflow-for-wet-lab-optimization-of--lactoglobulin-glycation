## Interpretation

The records contain prompts, questions, agent responses and merged summaries. They document how the workflow moved from project specification to tool screening, run-sheet construction, returned-result interpretation and final recommendation locking. They should not be interpreted as independent experimental evidence; wet-lab evidence is reported in the manuscript, Supplementary Tables and Source Data.
