\section*{Data and Code Repository Inventory}

The Supplementary Information contains the methods, extended figure and compact
tables needed to interpret the study. Complete machine-readable records and
analysis code are available in
\href{https://github.com/DudyLee/An-evidence-guided-multi-agent-workflow-for-wet-lab-optimization-of--lactoglobulin-glycation}{the project GitHub repository}.
The repository root contains a reviewer-facing \nolinkurl{README.md} and a
file-level \nolinkurl{FILE_MANIFEST.tsv} with file sizes and SHA-256 checksums.

\subsection*{Project data repository}

The project data repository contains the following data packages:
\begin{enumerate}
  \item \textbf{Structured literature evidence records.}
  Validated JSONL records, harmonized tabular records and reduction-field audit
  outputs for the literature-derived evidence layer. Key files include
  \nolinkurl{data/paper_data/Paper_run_data.computed_V2.jsonl},
  \nolinkurl{data/paper_data/Paper_run_data.computed_V2.selected_fields_table.csv},
  \nolinkurl{data/paper_data/reduction_compute_report.txt} and
  \nolinkurl{Knowledge_Extraction/literature_evidence_extraction_prompt.md}.

  \item \textbf{Condition-envelope and workflow-funnel support.}
  The grouped axis counts used for the Fig.~4a funnel and Supplementary
  Table~8 are provided in
  \nolinkurl{data/paper_data/condition_envelope_grouped_levels.csv}. Workflow
  accounting files are provided in
  \nolinkurl{protein_design/figures/workflow_analysis/workflow_word_counts.csv}
  and
  \nolinkurl{protein_design/figures/workflow_analysis/workflow_analysis_summary.json}.

  \item \textbf{Sequence-prior outputs.}
  The $\beta$-LG input sequence, BepiPred-3.0 residue-level output, NetGlycate
  lysine-level output and plotting-ready tables used for Fig.~2c,d. Key files
  include \nolinkurl{data/bepipred_data/rcsb_pdb_1BSQ.fasta},
  \nolinkurl{data/bepipred_data/bepipred3_results/raw_output.csv},
  \nolinkurl{data/netglycate_data/beta_lg_a.fasta} and
  \nolinkurl{data/netglycate_data/netglycate_results.csv}.

  \item \textbf{Run-sheet and identifier mapping.}
  Complete 24-condition plan, result-availability audit, formal R01--R24 mapping,
  source-workbook links and comparison-block definitions. Key files include
  \nolinkurl{data/paper_data/24_run_condition_result_matrix.xlsx},
  \nolinkurl{protein_design/tables/wetlab_runsheet/original_24_run_availability_audit.csv}
  and
  \nolinkurl{protein_design/tables/wetlab_runsheet/formal_24_run_result_mapping.csv}.

  \item \textbf{Raw and processed wet-lab data.}
  Raw assay workbooks, processed IgG-binding values, reduction calculations,
  replicate information and data dictionaries. The original workbook set is
  retained under \nolinkurl{data/raw_wetlab_workbooks/}. Processed files include
  \nolinkurl{protein_design/tables/glycation_results_organized/glycation_results_organized_2026-06-26.xlsx},
  \nolinkurl{protein_design/tables/glycation_results_organized/raw_cell_index.csv},
  \nolinkurl{protein_design/tables/glycation_results_organized/long_timecourse.csv}
  and
  \nolinkurl{protein_design/tables/glycation_results_organized/ribose_validation.csv}.

  \item \textbf{Agent prompts, meetings and locked-decision records.}
  Stage-level meeting records, aggregation outputs, prompt archive, wet-lab
  feedback packets and the locked pre-validation recommendation record. Key
  files include
  \nolinkurl{data/agent_prompts_meetings_decisions/agent_prompt_decision_archive.zip},
  \nolinkurl{data/agent_prompts_meetings_decisions/agent_prompt_decision_archive/prompt_archive_manifest.tsv}
  and
  \nolinkurl{data/agent_prompts_meetings_decisions/agent_prompt_decision_archive/locked_recommendation_record.md}.
\end{enumerate}

\subsection*{Project code repository}

The project code repository contains the Virtual Lab meeting framework
(\nolinkurl{src/virtual_lab/}), literature-extraction scripts and notes
(\nolinkurl{Knowledge_Extraction/}), analysis, data-organization and plotting
scripts (\nolinkurl{protein_design/scripts/}) and the Source Data workbook
builder
(\nolinkurl{protein_design/scripts/source_data_build/build_source_data_workbook.mjs}).
Package metadata and review dependencies are provided in
\nolinkurl{pyproject.toml} and \nolinkurl{requirements-review.txt}.

\section*{Source Data Inventory}

Source data are provided with this paper for quantitative main and
supplementary figures and are also included in the repository as
\nolinkurl{data/Source_Data.xlsx}. The Source Data workbook preserves panel
identifiers, plotted values, condition definitions, summary statistics,
replicate identifiers where applicable, error-bar definitions and links to the
corresponding raw or processed repository files.

\begin{enumerate}
  \item \textbf{Source Data for Fig. 2c,d.}
  BepiPred-3.0 residue scores, NetGlycate lysine scores, plotted positions,
  marker labels and score-processing fields, linked to
  \nolinkurl{data/bepipred_data/bepipred3_results/raw_output.csv} and
  \nolinkurl{data/netglycate_data/netglycate_results.csv}.

  \item \textbf{Source Data for Fig. 3a--c.}
  R01--R24 plotted IgG-binding reduction values, condition labels, error values,
  glucose marginal-gain calibration values, predicted range and final validation
  value, linked to
  \nolinkurl{protein_design/figures/wetlab_results_v2/Figure_3_v2_source_data.csv},
  \nolinkurl{protein_design/figures/wetlab_results_v2/Figure_3b_v2_matrix_source_data.csv}
  and
  \nolinkurl{protein_design/figures/wetlab_results_v2/Figure_3c_v2_source_data.csv}.

  \item \textbf{Source Data for Fig. 4a--c.}
  Funnel counts, phase-level interaction counts, role-level counts and counting
  rules used to generate the workflow-accounting panels, linked to
  \nolinkurl{data/paper_data/condition_envelope_grouped_levels.csv},
  \nolinkurl{protein_design/figures/workflow_analysis/workflow_word_counts.csv}
  and
  \nolinkurl{protein_design/figures/workflow_analysis/workflow_analysis_summary.json}.

  \item \textbf{Source Data for Supplementary Fig. 1.}
  Complete donor-time values, ultrasound and no-ultrasound arms, error values,
  donor labels, time points and source-workbook links, linked to
  \nolinkurl{protein_design/tables/glycation_results_organized/long_timecourse.csv}
  and
  \nolinkurl{protein_design/figures/supplementary_wetlab_landscape_options/landscape_options_manifest.csv}.
\end{enumerate}
